# Product Definition

## Project Name
Blood on the Clocktower Tracker

## Description
Ком'юніті сайт для трекінгу ігор Blood on the Clocktower, статистики гравців та каталогу ролей.

## Problem Statement
Немає зручного інструменту для відстеження результатів ігор BotC та статистики гравців, особливо українською мовою. Ком'юніті втрачає історію ігор та не може порівнювати результати гравців між сесіями.

## Target Users
Невелике ком'юніті друзів (~8-15 людей), які грають в BotC щонеділі. До 100 зареєстрованих користувачів максимум.

### User Roles
| Role | Description | Permissions |
|------|-------------|-------------|
| **Guest** | Незареєстрований відвідувач | Перегляд статистики, ігор, каталогу ролей |
| **Player** | Авторизований гравець | Додати себе на гру, обрати свою роль та результат |
| **Admin** | Адміністратор ком'юніті | Створення ігор, додавання будь-яких гравців, редагування/видалення всього, перезапис записів гравців |

## Key Goals
1. Зберігати історію всіх ігор з ролями та результатами
2. Надати гравцям персональну статистику (win rate, MVP, улюблені ролі)
3. Створити повний каталог ролей українською (~154 ролей з описами UA + EN)

## Key Features
- **Каталог ролей** — всі офіційні ролі BotC з описами українською та англійською
- **Трекінг ігор** — створення ігрових сесій з гравцями, ролями, результатами
- **Статистика гравців** — win rate, MVP count, улюблені ролі, серії перемог
- **Лідерборд** — рейтинги за перемогами, win %, MVP, окремо good/evil
- **Self-service** — гравці можуть додавати себе на ігри та обирати свої ролі
