## Summary

Сторінка каталогу ролей BotC з гридом всіх 160 ролей, фільтрацією по типу та виданню, пошуком по імені (UA/EN) та діалогом деталей ролі.

## Context

Blood on the Clocktower Tracker має 160 ролей у базі Supabase з українськими та англійськими назвами/описами. Каталог ролей — основна довідкова сторінка для ком'юніті.

## User Story

As a guest or player, I want to browse all BotC roles with filtering by type/edition and search by name, so that I can explore the role catalog in Ukrainian and English.

- [ ] `/roles` page shows all 160 roles in a responsive grid
- [ ] Filter by type (townsfolk, outsider, minion, demon, traveller, fabled) works
- [ ] Filter by edition (TB, BMR, SNV, experimental, KS, base3) works
- [ ] Search by name (UA or EN) works
- [ ] Click on role opens detail dialog with full description (UA + EN), type, edition, image
- [ ] Role cards show image, name (UA + EN format), type badge with correct color coding
- [ ] Mobile responsive (2 columns on mobile, 4-6 on desktop)
