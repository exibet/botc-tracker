## Summary

Games list, detail, create/edit pages with composable for full game session management.

## Context

The DB schema (games + game_players tables) and RLS policies are already in place. Auth with admin middleware is ready. The games page currently shows a placeholder.

## User Story

As an admin, I want to create and edit game sessions with players and roles.
As any visitor, I want to browse games and view game details with player tables.
